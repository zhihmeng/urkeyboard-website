基本設定

如何啟用 UrKeyboard

使用 UrKeyboard 之前，需要先在系統設定中啟用它。

操作步驟

  1. 打開手機的「設定」應用程式
  2. 前往「一般管理」或「語言與輸入法」
  3. 點擊「螢幕鍵盤」或「虛擬鍵盤」
  4. 點擊「管理鍵盤」
  5. 找到「UrKeyboard」並將開關打開
  6. 系統會彈出安全性提示，點擊「確定」

注意事項

  • 不同 Android 版本的設定路徑可能略有不同
  • 啟用後，還需要將 UrKeyboard 設為預設輸入法才能使用
  • 若找不到選項，可嘗試在設定中搜尋「輸入法」

如何設為預設輸入法

啟用後，還需要將 UrKeyboard 設定為預設輸入法，才能在所有 App 中使用。

操作步驟

  1. 打開手機的「設定」應用程式
  2. 前往「一般管理」→「語言與輸入法」
  3. 點擊「預設鍵盤」或「目前鍵盤」
  4. 從清單中選擇「UrKeyboard」

快速切換

您也可以在任何輸入框中，長按空白鍵或點擊輸入框右下角的鍵盤圖示來切換輸入法。

注意事項

  • 若未看到 UrKeyboard，請先確認已完成「啟用」步驟
  • 每次重新啟動後，設定都會保留

如何載入輸入法字典

UrKeyboard 支援 .cin 格式的輸入法字典。首次使用時必須載入至少一個字典才能開始打字。

什麼是字典？

字典（.cin 檔案）是一個「字根 → 中文字」的對應表，定義了您按下哪些按鍵可以輸出哪些中文字。不同輸入法（注音、行列、倉頡等）使用不同的字典。

從列表選擇（推薦）

  1. 開啟 UrKeyboard 應用程式
  2. 進入「設定」→「輸入法字典選擇」
  3. 從清單中點擊您要使用的輸入法名稱
  4. 等待字典載入完成（首次可能需要數秒）

手動匯入（進階）

  1. 取得 .cin 格式的字典檔案
  2. 使用檔案管理器將檔案複製到手機儲存空間
  3. 進入「設定」→「手動匯入字典」
  4. 找到並選擇您的 .cin 檔案

注意事項

  • 一次只能載入一個字典
  • 載入新字典後，舊字典會被取代
  • 若字典載入失敗，請確認 .cin 檔案格式正確

支援哪些輸入法字典

UrKeyboard 內建多種常用輸入法字典，也支援自訂 .cin 格式字典。

內建字典

輸入法 適合對象
注音 使用注音符號輸入的用戶（台灣最常見）
行列 使用行列 30 輸入法的用戶
全尺寸 喜好完整鍵盤佈局的用戶
五排 使用五排按鍵佈局的用戶
31 鍵 使用 31 鍵佈局的用戶

自訂字典

您也可以匯入符合 .cin 格式的自訂字典。.cin 格式是一種廣泛使用的輸入法定義格式，網路上有許多現成資源可下載。

如何更換輸入法字典

您可以隨時切換到不同的輸入法字典。

操作步驟

  1. 開啟 UrKeyboard 應用程式
  2. 進入「設定」→「輸入法字典選擇」
  3. 點擊您想切換的字典名稱
  4. 字典載入完成後，鍵盤即刻套用新字典

注意：切換字典後，目前正在輸入的內容會被清除。

重新顯示設定導覽

如果您想再次查看初次設定的引導流程，可以從設定頁面重新觸發。

操作步驟

  1. 開啟 UrKeyboard 應用程式
  2. 進入「設定」
  3. 在「其他與關於」分類中，點擊「重新顯示設定導覽」
  4. 設定導覽將從頭開始
輸入方式

中英文輸入切換

UrKeyboard 支援快速在中文模式與英文模式之間切換。

操作方法

  • 點擊「中/英」鍵：鍵盤左下角的模式切換鍵，點一下即可切換
  • 切換後，鍵盤上方的候選列會相應變化
  • 英文模式下，直接輸入拉丁字母

狀態辨識

可從「中/英」鍵的顯示文字判斷目前模式：

  • 顯示「中」→ 目前為中文輸入模式
  • 顯示「英」→ 目前為英文輸入模式

候選字的使用方式

輸入字根後，鍵盤上方的候選列會顯示對應的中文字或詞組。

基本操作

  • 點擊候選字：直接選取該字/詞輸出
  • 左右滑動候選列：瀏覽更多候選字，支援慣性滾動
  • 第一個候選字：通常以粗體顯示，為最常用的選擇

智慧連續輸入

開啟「連續輸入」功能後，UrKeyboard 會自動根據上下文斷詞，讓您不需要每個字單獨選取，大幅提升輸入速度。

候選列功能鍵

  • 剪貼簿圖示：開啟剪貼簿歷史
  • 詞庫頁籤：顯示自訂詞庫快速輸出

打繁出簡功能

「打繁出簡」讓您以繁體中文的方式輸入，但自動將輸出的文字轉換為簡體中文，方便對岸溝通。

啟用方法

  1. 在鍵盤上找到「簡」功能鍵
  2. 點擊後即啟用打繁出簡模式
  3. 再次點擊即關閉，回到繁體輸出

使用情境

  • 與中國大陸朋友或同事溝通
  • 填寫需要簡體中文的表單
  • 使用中國大陸的 App 時輸入文字

注意：此功能只轉換輸出的文字，您的輸入習慣（字根）不需要改變。

空白鍵游標控制

在空白鍵上滑動可以精確移動文字輸入游標，不需要放大鏡或精確點擊。

操作方法

  • 向左滑動空白鍵：游標向左移動一格
  • 向右滑動空白鍵：游標向右移動一格
  • 滑動速度越快，游標移動越快

使用場景

  • 需要修改長文字中間的錯字
  • 在小螢幕上精確定位游標
  • 不想中斷輸入流程的快速游標調整

提示：此功能在候選字選取狀態下同樣有效。

滑動選取文字

UrKeyboard 支援透過鍵盤上的滑動手勢來選取文字範圍，無需精確點擊文字。

操作方法

  1. 長按鍵盤任意按鍵
  2. 保持按壓並滑動
  3. 滑動方向決定選取範圍
  4. 放開手指完成選取

選取後的操作

  • 選取文字後可直接貼上替換
  • 系統剪貼簿工具列會出現複製/剪下選項
效率工具

剪貼簿歷史功能

UrKeyboard 會自動記錄您最近複製的內容，讓您隨時快速貼上之前複製過的文字。

開啟方式

  • 點擊候選列左側的剪貼簿圖示
  • 剪貼簿面板會展開在候選列下方

基本操作

  • 點擊項目：直接貼上該內容
  • 左右滑動：瀏覽更多剪貼簿記錄
  • 長按項目：顯示更多選項（加入詞庫、刪除等）

清除記錄

  • 長按單筆記錄選擇刪除
  • 或在剪貼簿面板中選擇「清除全部」

隱私提醒

剪貼簿記錄只儲存在您的裝置本機，UrKeyboard 不會上傳您的剪貼簿內容到伺服器。

自訂詞庫的使用

自訂詞庫讓您儲存常用的詞語、片語或文字模板，需要時一鍵輸出。

新增到詞庫

  1. 複製您想儲存的文字
  2. 開啟剪貼簿面板
  3. 長按剪貼簿中的項目
  4. 選擇「加入詞庫」

使用詞庫輸出

  1. 點擊候選列上的「詞庫」頁籤
  2. 瀏覽或搜尋您想輸出的詞條
  3. 點擊詞條即可直接輸出

管理詞庫

  • 前往「設定」→「自定義詞庫」
  • 可以新增、編輯或刪除詞條
  • 支援快捷碼（輸入特定字根後自動出現）

表情符號（Emoji）輸入

UrKeyboard 內建 Emoji 面板，讓您快速瀏覽與輸入各種表情符號。

進入 Emoji 模式

  • 點擊鍵盤上的「繪」鍵（通常位於鍵盤角落）
  • Emoji 面板會取代鍵盤顯示

瀏覽與輸入

  • 分類瀏覽：左右滑動或點擊頂部分類圖示切換分類
  • 搜尋：輸入關鍵字（如「笑」、「心」）搜尋相關 Emoji
  • 點擊輸入：點擊任意 Emoji 即可輸入到目前游標位置

離開 Emoji 模式

  • 點擊鍵盤面板中的「ᴬʙᴄ」或返回按鍵
  • 點擊任意輸入框外的區域

分離鍵盤操作

分離鍵盤模式將鍵盤分為左右兩半，適合雙手握持平板或大螢幕手機時使用。

啟用分離模式

  • 點擊分離圖示：在鍵盤上找到分離/合併圖示並點擊
  • 捏合手勢：在鍵盤上以雙指向外展開（需在設定中啟用捏合手勢）

關閉分離模式

  • 點擊合併圖示：再次點擊分離/合併圖示
  • 捏合手勢：在鍵盤上以雙指向內合攏

設定選項

前往「設定」→「大螢幕與平板設定」可調整：

  • 是否啟用分離鍵盤
  • 是否啟用切換動畫
  • 是否啟用捏合手勢
外觀主題

內建主題介紹與切換

UrKeyboard 提供多套精心設計的內建主題，讓您的鍵盤更具個人風格。

內建主題

  • 極簡白：清爽白色基調，適合日常使用
  • 極簡黑：深色背景，省電護眼
  • 科技藍：藍色調，現代科技感
  • 自然綠：清新綠色，舒緩視覺
  • 粉櫻：粉色調，溫柔浪漫
  • 自動跟隨系統：根據手機亮/暗色模式自動切換

切換主題

  1. 開啟 UrKeyboard 應用程式
  2. 進入「設定」→「主題風格」
  3. 點擊任一主題即可預覽並套用

主題編輯器使用指南

主題編輯器讓您自由調整鍵盤的每個視覺細節，打造專屬的個人化外觀。

進入主題編輯器

  1. 進入「設定」→「主題風格」
  2. 選擇一個主題作為基礎
  3. 點擊「編輯」或「自訂」按鈕

可調整的項目

  • 背景色：鍵盤整體背景顏色
  • 按鍵色：每個按鍵的顏色
  • 文字色：按鍵上文字的顏色
  • 背景圖：為鍵盤設定自訂背景圖片
  • 字型：按鍵文字的字體

儲存主題

  • 編輯完成後點擊「儲存」
  • 可為主題命名方便識別
  • 儲存的主題會出現在主題選擇列表中

主題商店與社群主題

主題商店收錄了社群設計師分享的精美主題，讓您的鍵盤與眾不同。

瀏覽與下載主題

  1. 進入「設定」→「主題風格」→「主題商店」
  2. 瀏覽社群主題列表
  3. 點擊主題可預覽效果
  4. 點擊「下載並套用」即可使用

Pro 功能

下載社群主題需要 Pro 訂閱。訂閱後可無限下載主題商店中的所有主題。

分類瀏覽

  • 可依「最新」、「最熱門」等條件篩選
  • 支援標籤分類（如：深色系、繽紛、簡約等）

字型選擇

為鍵盤按鍵選擇不同的字型，打造更具個性的輸入體驗。

進入字型選擇器

  1. 進入「設定」→「主題風格」→「主題編輯器」
  2. 點擊「字型」選項

選擇字型

  • 瀏覽可用字型列表
  • 點擊任一字型可預覽效果
  • 確認後點擊「套用」

打字音效與震動設定

調整按鍵的音效與震動回饋，讓打字體驗更符合您的偏好。

音效設定

  1. 進入「設定」→「主題風格」→「主題編輯器」
  2. 點擊「音效」選項
  3. 從列表中選擇喜歡的按鍵音效
  4. 點擊音效名稱可試聽
  5. 若不想要音效，選擇「靜音」

震動設定

  1. 進入「設定」→「輸入行為」
  2. 找到「按鍵震動回饋」開關
  3. 開啟或關閉震動回饋
Pro 功能

雲端同步功能說明

Pro 訂閱用戶可透過 Google 帳號將設定、詞庫與學習記錄同步到雲端，換機時輕鬆還原。

同步的項目

  • 鍵盤設定：主題、字型、音效等個人化設定
  • 自訂詞庫：您新增的所有詞條
  • 學習記錄：按鍵習慣的學習資料

啟用雲端同步

  1. 確認已訂閱 Pro 版
  2. 進入「設定」→「會員與雲端服務」→「雲端同步」
  3. 點擊「使用 Google 帳號登入」
  4. 選擇您的 Google 帳號並授權
  5. 等待首次同步完成

換機還原

  1. 在新手機安裝 UrKeyboard
  2. 訂閱或恢復 Pro 訂閱
  3. 以相同的 Google 帳號登入雲端同步
  4. 點擊「從雲端還原」

如何訂閱 Pro 版

Pro 訂閱為加值服務，解鎖雲端同步、所有社群主題下載等進階功能。

訂閱方案

  • 月繳：NT$30/月（隨時可取消）
  • 年繳：NT$240/年（省 33%）

訂閱步驟

  1. 進入「設定」→「會員與雲端服務」→「Pro 訂閱」
  2. 選擇訂閱方案（月繳或年繳）
  3. 點擊「立即訂閱」
  4. 透過 Google Play 完成付款

取消訂閱

可透過 Google Play →「訂閱管理」隨時取消，取消後 Pro 功能在訂閱到期前仍可使用。

恢復 Pro 訂閱

換機或重新安裝後，若 Pro 訂閱未自動恢復，可手動觸發還原。

操作步驟

  1. 進入「設定」→「會員與雲端服務」→「Pro 訂閱」
  2. 點擊「恢復訂閱」按鈕
  3. 系統會向 Google Play 查詢您的訂閱狀態
  4. 確認訂閱有效後，Pro 功能即刻解鎖

注意事項

  • 需使用訂閱時所用的同一個 Google 帳號
  • 需連接網路
  • 若問題持續，請聯絡客服 simon@molioapp.com
常見問題

鍵盤不出現怎麼辦

如果點擊輸入框後鍵盤沒有出現，請依以下步驟排查。

步驟 1：確認已啟用

  1. 前往「設定」→「一般管理」→「語言與輸入法」→「管理鍵盤」
  2. 確認 UrKeyboard 的開關是開啟狀態

步驟 2：確認設為預設

  1. 前往「設定」→「語言與輸入法」→「預設鍵盤」
  2. 確認選擇了 UrKeyboard

步驟 3：嘗試切換輸入法

  • 點擊輸入框，然後長按空白鍵或點擊右下角鍵盤圖示
  • 從列表選擇 UrKeyboard

步驟 4：重新啟動

  • 強制關閉 UrKeyboard 應用程式後重新開啟
  • 若問題持續，嘗試重新啟動手機

若以上步驟都無法解決，請聯絡我們：simon@molioapp.com

輸入的字和我打的不一樣

如果候選字或輸出的文字不符合預期，可能有以下幾個原因。

可能原因 1：字典不符合輸入法

您載入的字典與您使用的輸入法不一致。例如使用注音輸入法，但載入的是行列字典。

解決方法：前往「設定」→「輸入法字典選擇」，選擇符合您輸入習慣的字典。

可能原因 2：字典未正確載入

字典檔案損壞或格式不正確。

解決方法：重新載入字典，或下載並重新匯入字典檔案。

可能原因 3：連續輸入模式影響

連續輸入模式下，UrKeyboard 會嘗試自動組詞，有時可能選擇非預期的詞。

解決方法：可在「設定」→「輸入行為」中關閉「連續輸入」。

如何備份我的設定

Pro 版：雲端同步（推薦）

Pro 訂閱用戶可使用雲端同步功能，自動將設定備份到 Google 帳號中。

詳見「雲端同步功能說明」文章。

一般版：本機備份

  • 自訂詞庫：目前需手動記錄詞條內容
  • 自訂主題：主題檔案存放於應用程式私有目錄，不易手動備份
  • 字典檔案：若使用自訂 .cin 檔，請備份該檔案

建議：訂閱 Pro 版以獲得完整的自動雲端備份功能。

更換手機後怎麼還原

使用雲端同步還原（Pro 版）

  1. 在新手機安裝 UrKeyboard
  2. 完成初次設定（啟用 → 預設 → 載入字典）
  3. 進入「設定」→「Pro 訂閱」，恢復 Pro 訂閱
  4. 進入「雲端同步」，以原 Google 帳號登入
  5. 點擊「從雲端還原」，等待完成

手動還原（一般版）

  1. 安裝 UrKeyboard 並完成初次設定
  2. 重新選擇您慣用的輸入法字典
  3. 重新設定主題、音效等個人化選項
  4. 將備份的自訂詞庫重新匯入

如何更新到最新版本

自動更新

若 Google Play 設定了自動更新，UrKeyboard 有新版本時會自動下載並安裝。建議保持自動更新開啟，以獲得最新功能與安全修正。

手動檢查更新

  1. 開啟 Google Play
  2. 搜尋「UrKeyboard」或前往您的應用程式庫
  3. 若有可用更新，會顯示「更新」按鈕
  4. 點擊更新即可

更新後

  • 您的設定與詞庫不受影響
  • 若有重大更新，建議重新載入字典以確保相容性
