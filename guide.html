基本設定

如何啟用 UrKeyboard

使用 UrKeyboard 之前，需要先在系統設定中新增並啟用它。

操作步驟

  1. 開啟 iPhone 的「設定」App
  2. 前往「一般」→「鍵盤」
  3. 點擊「鍵盤」（第一個項目）
  4. 點擊「新增鍵盤…」
  5. 在第三方鍵盤列表中找到「UrKeyboard」並點擊加入
  6. 回到鍵盤列表，點擊「UrKeyboard」
  7. 開啟「允許完整取用」

允許完整取用

開啟「允許完整取用」後，UrKeyboard 才能正常存取輸入法字典、音效、震動回饋等資源。系統會顯示安全性提示，點擊「允許」即可。

UrKeyboard 不會蒐集或上傳任何輸入內容。

注意事項

  • 若在列表中找不到 UrKeyboard，請確認 App 已正確安裝
  • 啟用後，在任意輸入框長按左下角的「🌐 地球」按鈕即可切換到 UrKeyboard
  • 若找不到地球按鈕，可嘗試重新啟動裝置

如何載入輸入法字典

UrKeyboard 使用 .cin 格式的輸入法字典來定義字根與中文字的對應關係。

什麼是輸入法字典？

輸入法字典（.cin 檔案）記錄了每一個中文字對應的輸入碼。例如注音字典記錄了「ㄇㄚ」對應「媽、馬、罵…」，嘸蝦米字典記錄了「ttn」對應「林」等。不同輸入法需要使用各自的字典。

自動載入（推薦）

UrKeyboard 在首次開啟 App 時會自動載入所有內建的輸入法字典，無需手動操作。您只需在設定中選擇要使用的輸入法即可。

  1. 開啟 UrKeyboard App
  2. 在「基本設定」區域，點擊「輸入法字典」
  3. 從清單中點擊您要使用的輸入法名稱（如：注音、嘸蝦米）
  4. 鍵盤會立即套用新字典

手動匯入（進階）

您也可以匯入自訂的 .cin 格式字典：

  1. 在 App 的「基本設定」中，點擊「手動匯入字典說明」
  2. 依照說明操作即可匯入自訂字典

注意事項

  • 一次只能使用一個字典
  • 載入新字典後，舊字典會被取代
  • 若字典載入失敗，請確認 .cin 檔案格式正確

支援哪些輸入法字典

UrKeyboard 內建多種常用輸入法字典，也支援自訂 .cin 格式字典。

內建字典

  • 注音 — 字音類，使用注音符號輸入（台灣最常見）
  • 注音（許氏） — 字音類，許氏鍵盤排列
  • 嘸蝦米 — 字形類，以英文字母組合拆字
  • 倉頡 — 字形類，三代/五代倉頡
  • 大易 — 字形類，大易輸入法
  • 行列 — 字形類，行列輸入法
  • 拼音 — 拼音類，漢語拼音輸入法

字音與字形的差別

  • 字音輸入法（如注音、拼音）：依據發音來輸入，例如輸入「ㄇㄚˇ」得到「馬」
  • 字形輸入法（如嘸蝦米、倉頡）：依據字的形狀拆解來輸入，例如嘸蝦米輸入「ttn」得到「林」

自訂字典

您也可以匯入符合 .cin 格式的自訂字典。.cin 格式是一種廣泛使用的輸入法定義格式，網路上有許多現成資源可下載。

鍵盤配置說明

「鍵盤配置」決定了按鍵在螢幕上的排列方式，與「輸入法字典」是兩個獨立的設定。同一個輸入法字典可以搭配不同的鍵盤配置使用。

可選的鍵盤配置

  • 標準配置 — 標準四排按鍵佈局，適合大多數使用者
  • 注音配置 — 為注音輸入法最佳化的按鍵排列
  • 全尺寸配置 — 較接近電腦鍵盤的完整佈局
  • 五排配置 — 五排按鍵佈局，含獨立數字列
  • 行列配置 — 專為行列輸入法設計的按鍵排列
  • 31 鍵配置 — 31 鍵精簡佈局

如何更換鍵盤配置

  1. 開啟 UrKeyboard App
  2. 在「基本設定」區域，點擊「鍵盤配置」
  3. 點擊想要使用的配置
  4. 鍵盤會立即套用新配置

配置選擇建議

  • 注音使用者：建議使用「注音配置」，按鍵排列最符合注音輸入習慣
  • 嘸蝦米/倉頡等字形使用者：建議使用「標準配置」
  • 需要獨立數字列：可選擇「五排配置」或「全尺寸配置」

如何更換輸入法字典

您可以隨時切換到不同的輸入法字典。

操作步驟

  1. 開啟 UrKeyboard App
  2. 在「基本設定」區域，點擊「輸入法字典」
  3. 點擊您想切換的字典名稱
  4. 字典載入完成後，鍵盤即刻套用新字典

系統會自動根據輸入法類型調整鍵盤配置。例如選擇注音會自動切換到注音配置。

注意： 切換字典後，目前正在輸入的內容會被清除。

重新顯示設定導覽

如果您想再次查看初次設定的引導流程，可以從設定頁面重新觸發。

操作步驟

  1. 開啟 UrKeyboard App
  2. 滑到最底部的「其他與關於」區域
  3. 點擊「重新顯示引導」
  4. 設定導覽將從頭開始
輸入方式

中／英／符輸入方式切換

UrKeyboard 支援在中文、英文與符號三種模式之間快速切換。

切換方式

透過候選列上方的功能模式按鈕進行切換：

  • 「中」：切換至中文輸入模式（注音、嘸蝦米等，依您載入的字典決定）
  • 「英」：切換至英文輸入模式，直接輸出英文字母
  • 「符」：切換至符號與數字鍵盤，方便輸入標點符號和數字

長按選擇附加字元

在中文或英文模式下，部分按鍵長按後會彈出附加字元選單（如數字、常用符號），滑動選擇即可快速輸入，無需切換到符號鍵盤。

狀態辨識

目前所在的模式按鈕會以醒目顏色標示，方便您隨時辨識輸入狀態。

中英混輸說明

在中文輸入模式下，您可以直接輸入英文字母與數字，無需先切換到英文模式。

使用方式

  • 在中文模式下直接輸入英文字母或數字，這些內容會暫存在組字區
  • 按下 Enter 鍵或切換語言鍵即可將英數內容直接送出

自動空白

啟用「中英混打自動空白」後（設定 → 輸入行為），中文與英數之間會自動插入空白，讓排版更美觀。

範例

輸入「今天用 Camera 拍了好多照片」時，打完「今天用」後直接輸入 Camera，按 Enter 送出英文，接著繼續輸入「拍了好多照片」即可。啟用自動空白後，中英文之間會自動插入空白。

相關設定

  • Enter 送出緩衝區：按 Enter 時自動送出未完成的英數文字
  • 切換語言送出緩衝區：切換中英文時自動送出未完成的英數文字

以上選項皆可在「設定」→「輸入行為」中調整。

候選字的使用方式

輸入字根後，鍵盤上方的候選列會顯示對應的中文字或詞組。

基本操作

  • 點擊候選字：直接選取該字/詞輸出
  • 左右滑動候選列：瀏覽更多候選字
  • 第一個候選字：通常為最常用的選擇

數字鍵選字

當候選列顯示時，如果您使用的輸入法未佔用數字鍵作為字根（例如嘸蝦米輸入法），可以直接按下數字鍵 1～0 來選取對應位置的候選字，操作更快速。

  • 候選字下方會標示對應的選字鍵編號
  • 按下 - 鍵或 = 鍵可翻頁瀏覽更多候選字

注意： 注音等字音輸入法由於數字鍵已佔用作為聲調輸入，因此無法用數字鍵直接選字。

聯想詞

選取候選字後，候選列會自動顯示聯想詞，提示可能的下一個字或詞組。點擊聯想詞即可快速輸入，大幅提升打字效率。

智慧連續輸入

開啟「連續輸入」功能後（設定 → 輸入行為），UrKeyboard 會自動根據上下文斷詞，讓您不需要每個字單獨選取，大幅提升輸入速度。

候選列功能模式介紹

候選列左側提供多個功能模式按鈕，點擊即可快速切換不同功能。

文字按鈕

  • 「中」：切換至中文輸入模式
  • 「英」：切換至英文輸入模式
  • 「符」：切換至符號鍵盤
  • 「繪」：開啟 Emoji 表情符號面板
  • 「顏」：開啟顏文字面板（如 (╯°□°)╯︵ ┻━┻）
  • 「標」：開啟特殊符號面板（如 ★、♫、→ 等）
  • 「同」：同音字查詢 — 查詢與目前候選字相同讀音的其他字（以相同注音讀音的字為準）
  • 「簡」：打繁出簡 — 開啟後以繁體輸入、自動轉換為簡體輸出

圖示按鈕

  • 📋 剪貼簿：開啟剪貼簿歷史與自訂詞庫面板
  • ⇔ 分離鍵盤：切換分離/合併鍵盤模式（僅 iPad 和大螢幕裝置）
  • ▼ 收起：隱藏候選列或收起面板

使用方式

點擊功能按鈕進入對應模式後，再點擊其他模式按鈕（如「中」或「英」）即可離開，回到文字輸入。

表情符號（Emoji）輸入

UrKeyboard 內建 Emoji 面板，讓您快速瀏覽與輸入各種表情符號。

進入 Emoji 模式

點擊候選列上的「繪」功能按鈕即可開啟 Emoji 面板。

瀏覽與輸入

  • 分類瀏覽：點擊底部分類圖示切換不同類別（表情、動物、食物、活動等）
  • 滑動瀏覽：在面板上左右滑動瀏覽更多 Emoji
  • 點擊輸入：點擊任意 Emoji 即可輸入
  • 最近使用：系統會記錄您最近使用的 Emoji，方便快速選取

離開 Emoji 模式

直接點擊候選列上的其他功能模式按鈕（如「中」或「英」）即可回到文字輸入。

打繁出簡功能

「打繁出簡」讓您以繁體中文的方式輸入，但自動將輸出的文字轉換為簡體中文，方便對岸溝通。

啟用方法

  1. 在候選列找到「簡」功能按鈕
  2. 點擊後即啟用打繁出簡模式
  3. 再次點擊即關閉，回到繁體輸出

您也可以在 UrKeyboard App 的「輸入行為」區域中開啟或關閉。

使用範例

使用注音輸入「ㄊㄞˊ ㄨㄢ」，啟用「簡」後會輸出「台湾」（簡體），關閉則輸出「台灣」（繁體）。

注意： 此功能只轉換輸出的文字，您的輸入習慣（字根）不需要改變。

空白鍵游標控制

在空白鍵上滑動可以精確移動文字輸入游標，不需要精確點擊螢幕上的文字。

操作方法

  • 向左滑動空白鍵：游標向左移動
  • 向右滑動空白鍵：游標向右移動
  • 滑動速度越快，游標移動越快

使用場景

  • 需要修改長文字中間的錯字
  • 在小螢幕上精確定位游標
  • 不想中斷輸入流程的快速游標調整

手勢與快捷鍵操作一覽

UrKeyboard 支援多種手勢與快捷鍵操作，幫助您更快速地完成輸入。以下分別列出軟體鍵盤手勢與外接實體鍵盤快捷鍵。

軟體鍵盤手勢

  • 鍵盤區域向上滑動 → 清空候選列：在鍵盤上從任意按鍵向上滑動，即可清空目前的候選列與輸入中的字根
  • 鍵盤區域向下滑動 → 收起鍵盤：在鍵盤上從任意按鍵向下滑動，即可收起鍵盤
  • 空白鍵左右滑動 → 移動游標：在空白鍵上左右滑動可精確移動文字游標（詳見「空白鍵游標控制」）
  • 按鍵長按滑動 → 選擇附加字元：部分按鍵長按後會彈出附加字元選單（如數字、符號），滑動選擇即可輸入
  • 雙指捏合 → 分離／合併鍵盤：在 iPad 等大螢幕裝置上，雙指向外展開可分離鍵盤，向內合攏可合併鍵盤（需在設定中啟用）
  • 候選列 ✕ 按鈕 → 清空候選列：點擊候選列右方的 ✕ 按鈕可清空候選列

外接實體鍵盤快捷鍵

  • Esc → 清空候選列：輸入中有候選字時按下 Esc 可清空候選列與輸入中的字根；無候選時交由系統處理
  • Backspace → 刪除字根／文字：刪除最後輸入的字根；無組字時刪除前一個字元
  • Enter → 送出／換行：將目前組字中的內容送出，或在無組字時換行
  • Space → 空白／聲調：注音模式組字中時作為一聲標記；其餘情況輸出空白

小提示

  • 滑動手勢需滑動一定距離才會觸發，輕觸按鍵不會誤觸
  • 候選列可左右滑動瀏覽更多候選字
  • 連接實體鍵盤時可啟用「實體鍵盤優化」自動隱藏螢幕鍵盤（設定 → 輸入行為）
效率工具

剪貼簿歷史功能

UrKeyboard 會自動記錄您最近複製的內容，讓您隨時快速貼上之前複製過的文字。

開啟方式

點擊候選列上的 📋（剪貼簿）圖示按鈕，剪貼簿面板即會展開。

基本操作

  • 點擊項目：直接貼上該內容
  • 左右滑動：瀏覽更多剪貼簿記錄
  • 長按項目：顯示更多選項（加入詞庫、刪除等）

隱私提醒

剪貼簿記錄只儲存在您的裝置本機，UrKeyboard 不會上傳您的剪貼簿內容到伺服器。

自訂詞庫的使用

自訂詞庫讓您儲存常用的詞語、片語或文字模板，並可設定快捷碼快速輸出。

新增到詞庫

方法一：從剪貼簿加入

  1. 複製您想儲存的文字
  2. 開啟剪貼簿面板（點擊 📋 剪貼簿圖示按鈕）
  3. 長按剪貼簿中的項目
  4. 選擇「加入詞庫」

方法二：從設定頁面新增

  1. 開啟 UrKeyboard App
  2. 在「基本設定」中選擇「自訂詞庫」
  3. 點擊右上角的「+」按鈕新增詞條
  4. 輸入詞條內容，並可選擇設定快捷碼
  5. 點擊「儲存」

使用快捷碼

設定快捷碼後，在中文或英文模式下輸入該快捷碼（如 addr），候選列首位會出現「⌨ 您的詞條內容」，點擊即可輸出。

管理詞庫

前往「設定」→「自訂詞庫」可瀏覽所有詞條、搜尋、編輯快捷碼或刪除詞條。

查碼功能（反查字根）

不確定某個字的字根怎麼拆？UrKeyboard 的查碼功能可以幫您反查任何字的輸入碼，方便學習與記憶。

使用方式

  1. 輸入您想查碼的字的任意讀音（注音）
  2. 點擊候選列上的「同」按鈕，進入同音字模式
  3. 在同音字列表中點選目標字
  4. 該字會被送出，同時候選列上方會顯示「查碼: ○○○」，顯示該字的字根組合

顯示說明

  • 查碼結果會以字根符號呈現（例如「查碼: 日月」），而非鍵盤上的英文字母
  • 查碼指示會短暫顯示在候選列上方，方便您記憶後繼續輸入

適用情境

  • 學習拆字：剛開始學習字形輸入法時，透過查碼了解每個字的拆法
  • 忘記字根：遇到不常用的字，快速確認正確的輸入碼
  • 教學用途：教導他人時，即時示範字根拆解方式

分離鍵盤操作

分離鍵盤模式將鍵盤分為左右兩半，適合雙手握持 iPad 或大螢幕裝置時使用。

啟用條件

分離鍵盤僅適用於螢幕較大的裝置：

  • iPad：支援分離
  • iPhone 橫向模式：部分較大螢幕的 iPhone 在橫向持握時支援分離
  • 一般 iPhone 直向：螢幕寬度不足，分離後按鍵會過小，因此分離功能不會生效

操作方式

  • 點擊 ⇔ 分離鍵盤圖示按鈕：候選列上的分離鍵盤圖示按鈕可切換分離模式
  • 捏合手勢：在鍵盤上以雙指向外展開開啟分離，向內合攏關閉分離（需在設定中啟用）

  • 是否啟用分離式鍵盤
  • 是否啟用切換動畫
  • 是否啟用捏合手勢

輸入的字和我打的不一樣

如果候選字或輸出的文字不符合預期，可能有以下幾個原因。

可能原因 1：字典不符合輸入法

您選擇的字典與您使用的輸入法不一致。例如使用注音輸入法的習慣，但載入的是嘸蝦米字典。

解決方法： 開啟 UrKeyboard App，在「基本設定」→「輸入法字典」中，選擇符合您輸入習慣的字典。

可能原因 2：打繁出簡已開啟

如果看到簡體字，可能是「打繁出簡」功能已開啟。

解決方法： 點擊候選列上的「簡」按鈕關閉，或在 App 的「輸入行為」中關閉「打繁出簡」。

可能原因 3：連續輸入模式影響

連續輸入模式下，UrKeyboard 會嘗試自動組詞，有時可能選擇非預期的詞。

解決方法： 可在「輸入行為」中關閉「啟用連續輸入」。

可能原因 4：鍵盤配置不匹配

使用的鍵盤配置與輸入法不一致，導致按鍵對應的字根不正確。

解決方法： 在「基本設定」→「鍵盤配置」中選擇匹配的配置。

所有核心輸入功能（中文輸入、候選字、連續輸入、剪貼簿、自訂詞庫、聯想字詞等）完全免費，不需要 Pro。如果您是舊版加購使用者，已購買的功能（鍵盤高度、音效等）也永久可用。不訂閱 Pro 只是無法使用主題編輯器完整版、主題商店和雲端同步。

原版加購功能設定說明

如果您曾購買過舊版的加購功能，設定頁面會自動出現「原版加購功能」區塊，讓您繼續使用已購買的功能。

各項功能說明

  • 鍵盤高度調整：拖動滑桿調整鍵盤整體高度（80%～130%），所有按鍵會等比例縮放以填滿畫布
  • 按鍵文字大小：拖動滑桿調整按鍵上文字的字號（12pt～28pt）
  • 按鍵大小調整：拖動滑桿調整按鍵之間的間距（2pt～10pt），間距越小按鍵越大
  • 打字音效：點擊進入音效選擇頁面，可從「電腦打字音效」（15 種）和「打字機音效」（24 種）中選擇一個喜歡的按鍵聲音，點擊即可試聽
  • 聯想詞功能：此功能已免費開放給所有使用者，可在「輸入行為」區塊中開關
  • 單手模式：請使用 iOS 系統內建的單手鍵盤功能（長按 🌐 按鈕選擇靠左或靠右）

覆蓋主題設定

「覆蓋主題設定」開關決定原版加購功能的設定值是否會覆蓋目前使用的主題中相同項目的設定值：

  • 開啟：以「原版加購功能」區塊中的數值為準（鍵盤高度、按鍵文字大小、按鍵間距、打字音效），即使主題中也有設定這些項目，仍以此處的值優先
  • 關閉（預設）：以目前使用的主題設定值為準，此處的調整僅作為備用記錄

調整後何時生效？

所有調整皆即時生效，無需重新啟動鍵盤。滑桿拖動或音效選擇後，下次按鍵即可感受到變化。

恢復舊版購買

若設定頁面未出現「原版加購功能」區塊，可能是舊版購買記錄尚未偵測到。請前往「其他與關於」→「恢復舊版購買」手動驗證購買記錄。驗證時需使用與當初購買時相同的 Apple ID。
